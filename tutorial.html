第一课：OpenClaw 安装步骤

从零开始，一步步搭建你的 AI 助手

📋 系统要求

  • 操作系统：Linux / macOS / Windows (WSL2)
  • Node.js：≥ 18.0.0
  • 内存：≥ 2GB RAM
  • 硬盘：≥ 10GB 可用空间

🚀 步骤 1：安装 Node.js

# Ubuntu/Debian
sudo apt update
sudo apt install nodejs npm -y

# macOS
brew install node

# 验证安装
node --version
npm --version

🔧 步骤 2：安装 OpenClaw

# 全局安装
npm install -g openclaw

# 验证安装
openclaw --version

⚙️ 步骤 3：初始化配置

# 初始化
openclaw init

# 配置 API Key
nano ~/.openclaw/config.json

# 启动 Gateway
openclaw gateway start

🎯 步骤 4：安装技能

# 安装 ClawHub
npm install -g clawhub

# 安装推荐技能
clawhub install memory-master
clawhub install openclaw-security-scanner
clawhub install weather

📱 步骤 5：连接 Telegram

  1. 在 Telegram 搜索 @BotFather
  2. 发送 /newbot 创建 bot
  3. 保存 Bot Token
  4. 编辑 ~/.openclaw/config.json 添加 Telegram 配置
  5. 重启 Gateway：openclaw gateway restart
🎉 完成！ 你已经成功安装了 OpenClaw。
