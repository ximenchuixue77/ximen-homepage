1.5 连接软件

连接 Telegram、飞书、QQ、WhatsApp

📱 支持的通讯软件

OpenClaw 支持多种通讯软件，选择你常用的平台连接：

  • Telegram - 最推荐，功能最完整
  • 飞书 - 企业办公首选
  • QQ - 国内用户友好
  • WhatsApp - 国际用户常用

🤖 连接 Telegram

步骤 1：创建 Bot

  1. 在 Telegram 搜索 @BotFather
  2. 发送 /newbot
  3. 按提示设置 Bot 名称和用户名
  4. 保存 Bot Token（格式：123456:ABC-DEF...）

步骤 2：获取 User ID

  1. 在 Telegram 搜索 @userinfobot
  2. 发送任意消息
  3. 保存你的 User ID（纯数字）

步骤 3：配置 OpenClaw

# 编辑配置文件
nano ~/.openclaw/config.json

# 添加 Telegram 配置：
{
  "telegram": {
    "token": "123456:ABC-DEF-your-bot-token",
    "allowedUsers": [123456789]
  }
}

# 重启 Gateway
openclaw gateway restart

🚀 连接飞书

步骤 1：创建飞书应用

  1. 访问 飞书开放平台
  2. 创建企业自建应用
  3. 获取 App ID 和 App Secret
  4. 配置机器人权限和回调地址

步骤 2：配置 OpenClaw

# 编辑配置文件
nano ~/.openclaw/config.json

# 添加飞书配置：
{
  "feishu": {
    "appId": "your-app-id",
    "appSecret": "your-app-secret"
  }
}

# 重启 Gateway
openclaw gateway restart

💬 连接 QQ

步骤 1：安装 go-cqhttp

# 下载 go-cqhttp
wget https://github.com/Mrs4s/go-cqhttp/releases/latest/download/go-cqhttp_linux_amd64.tar.gz
tar -xzf go-cqhttp_linux_amd64.tar.gz

# 运行并配置
./go-cqhttp

步骤 2：配置 OpenClaw

# 编辑配置文件
nano ~/.openclaw/config.json

# 添加 QQ 配置：
{
  "qq": {
    "host": "127.0.0.1",
    "port": 5700,
    "allowedUsers": [123456789]
  }
}

# 重启 Gateway
openclaw gateway restart

📞 连接 WhatsApp

步骤 1：扫码登录

# 启动 WhatsApp 连接
openclaw whatsapp connect

# 会显示二维码，用 WhatsApp 扫码登录

步骤 2：验证连接

# 查看连接状态
openclaw whatsapp status

# 发送测试消息
openclaw whatsapp send "Hello from OpenClaw!"
✅ 连接完成！
选择你常用的平台连接，就可以随时随地和 AI 助手对话了。
