1.3 初始化配置

创建工作空间并配置 API Key

📁 创建工作空间

# 初始化 OpenClaw
openclaw init

# 这会创建以下目录结构：
# ~/.openclaw/
# ├── workspace/      # 工作空间（存放记忆、技能等）
# ├── config.json     # 主配置文件
# └── gateway.json    # 网关配置

🔑 配置 API Key

步骤 1：获取 API Key

  • Anthropic Claude：console.anthropic.com
  • OpenAI：platform.openai.com

步骤 2：编辑配置文件

# 编辑配置文件
nano ~/.openclaw/config.json

# 添加你的 API Key：
{
  "models": {
    "default": "claude-sonnet-4",
    "apiKey": "sk-ant-your-api-key-here"
  }
}

🚀 启动 Gateway

# 启动 OpenClaw Gateway（后台服务）
openclaw gateway start

# 查看运行状态
openclaw gateway status

# 查看实时日志
openclaw gateway logs -f
✅ 配置完成！
Gateway 启动成功后，OpenClaw 就可以正常工作了。
