⚠️ 关键注意事项

1. 电话号码

  • 必须是全新的号码
  • 如果号码已使用，电话1字段会为空
  • 测试号码范围：13900000001-13900000099

2. 文本输入

  • 所有文本输入框必须先清空再输入
  • 使用 fill('') 清空，然后 type() 输入
  • 输入延迟：100ms

3. 下拉选择

  • 点击输入框 → 输入文字 → 等待3秒 → 点击第一个选项
  • 等待时间不能少于3秒

4. 商品勾选

  • 必须根据价格判断（/\d{3,}/.test(text)）
  • 必须点击第一列的复选框（td:first-child input[type="checkbox"]）
  • 不能点击其他位置

5. 确定按钮

  • 选择器：.ant-modal-footer .ant-btn-primary
  • 超时时间：10秒
  • 这是最准确的选择器，必须优先使用
