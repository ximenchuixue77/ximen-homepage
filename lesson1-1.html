1.1 环境准备

安装 OpenClaw 之前，先准备好运行环境

📋 系统要求

  • 操作系统：Linux / macOS / Windows (WSL2)
  • Node.js：≥ 18.0.0
  • 内存：≥ 2GB RAM
  • 硬盘：≥ 10GB 可用空间
  • 网络：稳定的互联网连接

🔧 安装 Node.js

Ubuntu/Debian：

sudo apt update
sudo apt install nodejs npm -y

# 验证安装
node --version
npm --version

macOS：

brew install node

# 验证安装
node --version
npm --version

Windows (WSL2)：

# 先安装 WSL2，然后按 Ubuntu 步骤操作

✅ 验证环境

确保 Node.js 版本 ≥ 18.0.0：

node --version
# 应该显示：v18.x.x 或更高
