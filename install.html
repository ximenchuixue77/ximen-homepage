⚔️ OpenClaw 安装教程

从零开始，打造你的 AI 助手

📋 系统要求

  • 操作系统：Linux / macOS / Windows (WSL2)
  • Node.js：≥ 18.0.0
  • 内存：≥ 2GB
  • 硬盘：≥ 10GB 可用空间

🚀 快速安装

1. 安装 Node.js

# Ubuntu/Debian sudo apt update sudo apt install nodejs npm -y # macOS (使用 Homebrew) brew install node # 验证安装 node --version npm --version

2. 安装 OpenClaw

# 全局安装 npm install -g openclaw # 验证安装 openclaw --version
✅ 安装成功！ 如果看到版本号，说明 OpenClaw 已经安装完成。

⚙️ 初始化配置

1. 创建工作空间

# 初始化 OpenClaw openclaw init # 这会创建以下目录结构： # ~/.openclaw/ # ├── workspace/ # 工作空间 # ├── config.json # 配置文件 # └── gateway.json # 网关配置

2. 配置 API Key

# 编辑配置文件 nano ~/.openclaw/config.json # 添加你的 API Key： { "models": { "default": "claude-sonnet-4", "apiKey": "your-api-key-here" } }
💡 提示： 你可以使用 Anthropic、OpenAI、或其他兼容的 API。

3. 启动 Gateway

# 启动 OpenClaw Gateway openclaw gateway start # 查看状态 openclaw gateway status # 查看日志 openclaw gateway logs

🔧 安装技能 (Skills)

1. 安装 ClawHub CLI

npm install -g clawhub

2. 搜索并安装技能

# 搜索技能 clawhub search memory # 安装技能 clawhub install memory-master # 查看已安装技能 clawhub list

3. 推荐技能

  • memory-master：精准记忆系统
  • openclaw-security-scanner：安全扫描工具
  • weather：天气查询
  • gog：Google Workspace 集成

🔒 安全配置

⚠️ 重要： 安装完成后，务必运行安全扫描！
# 安装安全扫描工具 clawhub install openclaw-security-scanner # 运行扫描 bash ~/.openclaw/workspace/skills/openclaw-security-scanner/oc-security-scan.sh # 自动修复问题 bash ~/.openclaw/workspace/skills/openclaw-security-scanner/oc-security-scan.sh --fix

📱 连接 Telegram

1. 创建 Telegram Bot

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot 创建新 bot
  3. 按提示设置名称和用户名
  4. 保存 Bot Token

2. 配置 OpenClaw

# 编辑配置 nano ~/.openclaw/config.json # 添加 Telegram 配置： { "telegram": { "token": "your-bot-token", "allowedUsers": [123456789] } }

3. 重启 Gateway

openclaw gateway restart

💾 备份与恢复

备份工作空间

# 手动备份 tar -czf openclaw-backup-$(date +%Y%m%d).tar.gz ~/.openclaw/workspace # 自动备份（添加到 crontab） 0 2 * * * tar -czf ~/backups/openclaw-$(date +\%Y\%m\%d).tar.gz ~/.openclaw/workspace

恢复工作空间

# 解压备份 tar -xzf openclaw-backup-20260311.tar.gz -C ~/ # 重启 Gateway openclaw gateway restart

🐛 常见问题

Q: 安装失败，提示权限错误？

# 使用 sudo 安装 sudo npm install -g openclaw # 或者修改 npm 全局目录 mkdir ~/.npm-global npm config set prefix '~/.npm-global' echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc source ~/.bashrc

Q: Gateway 启动失败？

# 查看详细日志 openclaw gateway logs --tail 100 # 检查端口占用 lsof -i :18789 # 修改端口 nano ~/.openclaw/gateway.json

Q: 如何更新 OpenClaw？

# 更新到最新版本 npm update -g openclaw # 更新技能 clawhub update --all
