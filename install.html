📋 系统要求

  • 操作系统：Linux / macOS / Windows (WSL2)
  • Node.js：≥ 18.0.0
  • 内存：≥ 2GB RAM
  • 硬盘：≥ 10GB 可用空间
  • 网络：稳定的互联网连接

🚀 步骤 1：环境准备

安装 Node.js

# Ubuntu/Debian sudo apt update sudo apt install nodejs npm -y # macOS (使用 Homebrew) brew install node # 验证安装 node --version npm --version
💡 提示： 如果 Node.js 版本低于 18，建议使用 nvm 安装最新版本。

🔧 步骤 2：安装 OpenClaw

全局安装

# 使用 npm 全局安装 npm install -g openclaw # 验证安装 openclaw --version
✅ 安装成功！ 如果看到版本号（如 2026.3.8），说明 OpenClaw 已经安装完成。

⚙️ 步骤 3：初始化配置

创建工作空间

# 初始化 OpenClaw openclaw init # 这会创建以下目录结构： # ~/.openclaw/ # ├── workspace/ # 工作空间（存放记忆、技能等） # ├── config.json # 主配置文件 # └── gateway.json # 网关配置

配置 API Key

# 编辑配置文件 nano ~/.openclaw/config.json # 添加你的 API Key： { "models": { "default": "claude-sonnet-4", "apiKey": "sk-ant-your-api-key-here" } }
ℹ️ API Key 获取：
  • Anthropic Claude： console.anthropic.com
  • OpenAI： platform.openai.com

启动 Gateway

# 启动 OpenClaw Gateway（后台服务） openclaw gateway start # 查看运行状态 openclaw gateway status # 查看实时日志 openclaw gateway logs -f

🎯 步骤 4：安装技能 (Skills)

安装 ClawHub CLI

# 安装技能管理工具 npm install -g clawhub # 验证安装 clawhub --version

推荐技能安装

🧠 memory-master - 精准记忆系统

结构化记忆管理，自动索引，启发式召回

clawhub install memory-master

🔒 openclaw-security-scanner - 安全扫描

检查配置安全性，自动修复常见问题

clawhub install openclaw-security-scanner

🌤️ weather - 天气查询

获取实时天气和预报信息

clawhub install weather
⚠️ 安全提醒： 安装技能后，务必运行安全扫描！
bash ~/.openclaw/workspace/skills/openclaw-security-scanner/oc-security-scan.sh --fix

📱 步骤 5：连接 Telegram（可选）

创建 Telegram Bot

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot 创建新 bot
  3. 按提示设置名称和用户名
  4. 保存 Bot Token（格式：123456:ABC-DEF...）
  5. 获取你的 User ID（可以用 @userinfobot）

配置 OpenClaw

# 编辑配置文件 nano ~/.openclaw/config.json # 添加 Telegram 配置： { "telegram": { "token": "123456:ABC-DEF-your-bot-token", "allowedUsers": [123456789] } }

重启 Gateway

openclaw gateway restart
🎉 完成！ 现在可以在 Telegram 中和你的 AI 助手对话了！

💾 备份与恢复

手动备份

# 备份整个工作空间 tar -czf openclaw-backup-$(date +%Y%m%d).tar.gz ~/.openclaw/workspace # 备份到指定目录 tar -czf ~/backups/openclaw-$(date +%Y%m%d).tar.gz ~/.openclaw/workspace

自动备份（推荐）

# 添加到 crontab（每天凌晨 2 点自动备份） crontab -e # 添加以下行： 0 2 * * * tar -czf ~/backups/openclaw-$(date +\%Y\%m\%d).tar.gz ~/.openclaw/workspace

恢复备份

# 解压备份文件 tar -xzf openclaw-backup-20260311.tar.gz -C ~/ # 重启 Gateway openclaw gateway restart

🐛 常见问题

Q: 安装失败，提示权限错误？

# 方案 1：使用 sudo sudo npm install -g openclaw # 方案 2：修改 npm 全局目录（推荐） mkdir ~/.npm-global npm config set prefix '~/.npm-global' echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc source ~/.bashrc npm install -g openclaw

Q: Gateway 启动失败？

# 查看详细日志 openclaw gateway logs --tail 100 # 检查端口占用 lsof -i :18789 # 如果端口被占用，修改配置 nano ~/.openclaw/gateway.json # 修改 "port": 18789 为其他端口

Q: 如何更新 OpenClaw？

# 更新 OpenClaw 本体 npm update -g openclaw # 更新所有技能 clawhub update --all # 重启 Gateway openclaw gateway restart

Q: 忘记 API Key 怎么办？

# 查看当前配置 cat ~/.openclaw/config.json # 重新编辑配置 nano ~/.openclaw/config.json
